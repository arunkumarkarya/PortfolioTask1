# 🌐 Personal Portfolio Website

A responsive personal portfolio website built using **HTML** and **CSS** to showcase my profile, skills, projects, and contact information.

## 🚀 Features

- 🏠 Home section with introduction
- 👤 About Me section
- 💻 Skills section
- 📂 Projects section
- 📩 Contact/Message form
- 📄 Resume download option
- 🔗 Social media links
- 📱 Responsive and clean UI

## 🛠️ Technologies Used

- HTML5
- CSS3
- Font Awesome Icons
